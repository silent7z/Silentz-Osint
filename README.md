# Silentz Osint

Un couteau suisse pour l'osint léger et rapide en Python pour centraliser simplement vos recherches de base.

### ⚡️ Fonctionnalités
- **Usernames** : Check rapide sur les réseaux sociaux principaux (Insta, GitHub, Twitter, etc.).
- **IP & Domains** : Récupération des infos WHOIS, ISP et géolocalisation.
- **Emails** : Vérification du format et des serveurs MX (avec lien direct vers les leaks).
- **Images** : Extracteur de métadonnées EXIF pour choper les infos cachées (modèle photo, coordonnées GPS si dispo).
